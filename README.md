## Описание проекта

Это графическое приложение (GUI), созданное на языке Python, которое позволяет искать пользователей на платформе GitHub с помощью официального **GitHub API**.

**Основные возможности программы:**
*   **Поиск:** Ввод поискового запроса (никнейм, имя) для нахождения пользователей.
*   **Отображение результатов:** Результаты выводятся в виде удобного списка.
*   **Избранное:** Возможность добавить пользователя в избранное.
*   **Сохранение данных:** Список избранных пользователей сохраняется локально в файл `data/favorites.json`.
*   **Валидация ввода:** Проверка на пустой запрос.
